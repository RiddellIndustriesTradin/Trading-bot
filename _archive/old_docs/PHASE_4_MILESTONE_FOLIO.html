⚙️ Process Steps

  1. Signal: Pine Script detects 3-layer confirmation (regime + entry + confirmation)
  2. Alert: TradingView fires webhook with JSON payload
  3. Validate: Bot checks asset, action, confidence, and schema
  4. Execute: Market order placed on Kraken ($10 AUD)
  5. Protect: Auto stop-loss (-2.0%) placed immediately
  6. Notify: Telegram message sent with entry + SL/TP targets
  7. Log: Trade recorded in JSONL + Markdown journal
